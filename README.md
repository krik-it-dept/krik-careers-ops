## Stack

Pure HTML + CSS, no build step. Fonts loaded from Google Fonts.

- `Fraunces` — display serif
- `Manrope` — body
- `JetBrains Mono` — monospace labels

## Edit & deploy

Open the page file in any editor. Push to `main`, GitHub Pages serves it within ~1 minute.

```bash
git add .
git commit -m "Update JD"
git push
```
